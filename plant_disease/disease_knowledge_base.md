### 2. 番茄细菌性斑点病 | Tomato Bacterial Spot
**类型**: 训练类别

| 项目 | 内容 |
|------|------|
| **病原** | 野油菜黄单胞菌番茄致病变种 (*Xanthomonas vesicatoria*) |
| **症状** | 叶片出现水渍状小斑点，后变为褐色至黑色，边缘有黄色晕圈，严重时斑点连片导致叶片枯死脱落。果实上形成疮痂状突起。 |
| **发病条件** | 高温高湿环境，暴风雨后易发病，种子带菌是主要初侵染源。 |
| **视觉特征** | 深褐色至黑色小斑点，黄色晕圈，水渍状边缘 |

**预防措施**:
- 选用无病种子和抗病品种
- 种子消毒处理（温汤浸种或药剂处理）
- 避免连作，实行2-3年轮作
- 控制田间湿度，避免大水漫灌
- 及时清除病株残体

**推荐农药**:
- 77%氢氧化铜可湿性粉剂 500倍液
- 72%农用链霉素可溶性粉剂 4000倍液
- 20%噻菌铜悬浮剂 500倍液
- 3%中生菌素可湿性粉剂 800倍液

### 3. 番茄早疫病 | Tomato Early Blight
**类型**: 训练类别

| 项目 | 内容 |
|------|------|
| **病原** | 茄链格孢菌 (*Alternaria solani*) |
| **症状** | 叶片上出现圆形或近圆形褐色病斑，具有明显的同心轮纹，形似靶心。病斑周围常有黄色晕圈，严重时叶片干枯脱落，由下部叶片向上蔓延。 |
| **发病条件** | 温暖潮湿环境（20-30°C，最适25-28°C），相对湿度80%以上，多雨季节，连作地块，植株衰弱时易发病。 |
| **视觉特征** | 同心轮纹状褐色斑点，靶心样病斑，黄色晕圈 |

**预防措施**:

- 实行3年以上轮作
- 加强田间管理，及时整枝打杈
- 控制氮肥用量，增施磷钾肥
- 及时清除病叶和田间病残体

**推荐农药**:
- 75%百菌清可湿性粉剂 600倍液
- 80%代森锰锌可湿性粉剂 500倍液
- 10%苯醚甲环唑水分散粒剂 1500倍液

### 4. 番茄晚疫病 | Tomato Late Blight
**类型**: 训练类别

| 项目 | 内容 |
|------|------|
| **病原** | 致病疫霉 (*Phytophthora infestans*) |
| **症状** | 叶片出现水渍状暗绿色病斑，迅速扩大变褐，湿度大时叶背可见白色霉层。茎部出现褐色条斑，果实表面出现云纹状褐色硬斑。病情发展迅速，可导致全株枯死。 |
| **发病条件** | 低温高湿环境（10-22°C，最适15-20°C，相对湿度>90%），多雨、多雾天气，昼夜温差大时易流行。 |
| **视觉特征** | 水渍状暗绿色斑块，白色霉层，快速蔓延 |

**预防措施**:
- 选用抗病品种
- 高垄栽培，地膜覆盖
- 合理密植，保持通风
- 避免大水漫灌，控制田间湿度
- 及时清除中心病株
- 与非茄科作物轮作

**推荐农药**:
- 68.75%氟吡菌胺·霜霉威悬浮剂 600倍液
- 72%霜脲·锰锌可湿性粉剂 600倍液

### 9. 番茄黄化曲叶病毒病 | Tomato Yellow Leaf Curl Virus ⭐零样本
**类型**: 🎯 零样本测试类别

| 项目 | 内容 |
|------|------|
| **病原** | 番茄黄化曲叶病毒 (TYLCV)，属双生病毒科菜豆金色花叶病毒属，由烟粉虱(B型)持久性传播 |
| **症状** | 植株矮化，生长点叶片黄化、变小、皱缩，叶缘上卷呈杯状或匙状。叶脉间褪绿发黄，叶片增厚变硬。花器发育不良，坐果困难。早期染病植株几乎绝收。 |
| **发病条件** | 高温干旱季节，烟粉虱大量发生时传播迅速。B型烟粉虱获毒后可终生传毒。保护地和露地均可发生，秋大棚发病最重。 |
| **视觉特征** | 叶片黄化上卷，皱缩变小，植株矮化 |

**预防措施**:
- 选用抗病或耐病品种（如申粉V1、浙粉702等）
- 培育无虫无病壮苗
- 设置40-60目防虫网阻隔烟粉虱
- 及时防治烟粉虱是关键措施
- 发现病株立即拔除销毁
- 清除田间杂草减少虫源
- 悬挂黄板诱杀烟粉虱成虫

**推荐农药**:
- 25%噻虫嗪水分散粒剂 2500倍液（治虫）
- 10%吡虫啉可湿性粉剂 1500倍液（治虫）
- 20%啶虫脒可溶性粉剂 2500倍液（治虫）
- 2%宁南霉素水剂 300倍液（抑制病毒）
- 0.5%菇类蛋白多糖水剂 300倍液（诱导抗性）

- 工具用火焰或消毒液消毒
- 发现病株及时拔除
- 避免吸烟者接触植株

| **病原** | 玉米尾孢菌 (*Cercospora zeae-maydis*) |
| **症状** | 叶片上出现长方形至不规则形灰褐色病斑，沿叶脉扩展，边缘平行于叶脉呈直线状。病斑可连片，严重时叶片枯死。湿度大时病斑表面可见灰色霉层。 |
| **发病条件** | 温暖高湿环境（25-30°C，相对湿度>80%），连作田、低洼积水地块发病重。 |
| **视觉特征** | 长方形灰褐色病斑，沿叶脉平行扩展 |

- 增施磷钾肥，提高植株抗性

**推荐农药**:

- 75%百菌清可湿性粉剂 600倍液

### 18. 玉米北方叶枯病 | Corn Northern Leaf Blight (NCLB) ⭐零样本

### 21. 葡萄黑麻疹病 | Grape Esca (Black Measles)
**类型**: 训练类别

| 项目 | 内容 |
|------|------|
| **病原** | 多种真菌复合侵染，包括 *Phaeomoniella chlamydospora*、*Phaeoacremonium* spp. 等 |
| **症状** | 叶片出现叶脉间褪绿黄化，逐渐变为红褐色或紫褐色，呈虎斑状花叶。严重时叶片焦枯脱落。果实上出现紫褐色斑点，果皮开裂。枝干横切面可见褐色或黑色坏死组织。慢性型多年发展，急性型可导致突然枯死。 |
| **发病条件** | 多年生葡萄园，伤口多（修剪口、嫁接口）、树龄大时发病重。病菌可从伤口侵入，在木质部蔓延。 |
| **视觉特征** | 叶脉间褪绿，虎斑状花叶，红褐色至紫褐色 |

**预防措施**:
- 修剪时避免大伤口，伤口涂保护剂
- 发现病株及时挖除或重度修剪
- 保持树势健壮，避免过度负载
- 嫁接时使用健康接穗
- 苗木检疫，避免引入带病苗

**推荐农药**:
- 目前无特效药剂
- 修剪伤口涂抹5%硫酸铜溶液消毒
- 可尝试注射杀菌剂（咨询专业人员）
- 重在预防和管理

### 25. 马铃薯晚疫病 | Potato Late Blight
**类型**: 训练类别

| 项目 | 内容 |
|------|------|
| **病原** | 致病疫霉 (*Phytophthora infestans*) |
| **症状** | 叶片出现水渍状暗绿色病斑，从叶尖或叶缘开始，迅速扩大变褐。湿度大时叶背面边缘产生白色霉层（孢子囊）。茎部出现褐色条斑。块茎表面出现褐色凹陷斑，内部呈铁锈色坏死。病情发展迅速，可造成全田毁灭。 |
| **发病条件** | 低温高湿环境（10-20°C，相对湿度>90%），连续阴雨、多雾时极易流行，是毁灭性病害。 |

- 高垄栽培，及时培土
- 避免大水漫灌，雨后及时排水
- 发现中心病株立即拔除并喷药
- 收获前割除地上部茎叶
- 与非茄科作物轮作
